Hangman Arcade

A modern Hangman game built with Python + Streamlit, featuring difficulty modes, word categories, hints, stats, leaderboards, and player analytics.

Features:

Virtual keyboard
Difficulty levels (Easy / Medium / Hard)
Multiple word categories
Hint system (costs lives)
Data stored in SQLite
Player stats & streak tracking
Leaderboard
Analytics dashboard
CSV export

Analytics Includes:

Win rate
Avg attempts
Win/Loss pie chart
Win rate by word length
Timeline progress
Recent games
Leaderboard rankings
